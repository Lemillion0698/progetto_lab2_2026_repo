libmr - Framework MapReduce in C
=================================
Autore: Mbayanga Kevin
Corso:  Laboratorio II - Universita di Pisa

COMPILAZIONE
  make          compila libmr.a e wordcount
  make test     compila ed esegue la suite di test automatizzati
  make clean    rimuove tutti i file generati

ESECUZIONE
  ./wordcount   esegue il word count su tests/input.txt

OUTPUT
  output        file binario con i risultati (token, conteggio)
  mr.log        file di log dell'esecuzione
  statistics.txt statistiche di ogni esecuzione
